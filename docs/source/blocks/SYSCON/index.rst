系统管理 (SYSCON)
=========================

概述
----
系统管理为整个芯片提供时钟源，包括系统时钟切换、外设时钟门控、工作模式选择以及版本控制等功能。还可通过单独时钟的开或关，时钟源选择来进行功耗控制。


特性
----
-  时钟控制

-  工作模式选择

-  端口唤醒设置

-  低电压复位LVR

-  可编程电压检测PVD

-  复位控制及状态

-  UID
