端口控制 (PORTCON) 
=====================

概述
----

端口控制模块主要包括管脚输入使能，管脚功能配置，I/O上拉、下拉、推挽、开漏配置等。

SWM221系列所有型号PORTCON模块操作均相同，部分型号无对应管脚时，对应寄存器位无效。

特性
----
-  配置I/O引脚为特定功能

-  支持上拉/下拉/推挽/开漏功能

-  配置管脚输入使能
