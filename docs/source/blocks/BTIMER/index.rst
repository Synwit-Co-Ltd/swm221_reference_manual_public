基础定时器 (BTIMER)
=====================

概述
----

基础计数器模块，SWM221系列所有型号BTIMER操作均相同，不同型号BTIMER数量可能不同。使用前需使能BTIMER模块时钟。

每个BTIMER模块均具备定时器功能，并具备一个8位分频器，每个定时器具备独立中断源。


特性
----

-  4路24位通用定时器

-  每路均具备独立8位预分频

-  独立中断源

-  PWM输出功能
