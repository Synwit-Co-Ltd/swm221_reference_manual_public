除法器 (DIV)
============ 

概述
----
在需要高精度计算的应用，硬件除法器能加快除法运算，提高系统效率。SWM221系列所有型号除法器模块操作均相同。使用前需使能除法器模块时钟。

特性
----
-  支持32位整数除法运算及求余运算

-  支持32位开方运算，支持小数位

-  除法单次运算耗时8个时钟，不包括读写寄存器时间

-  开方单次运算耗时16/32个时钟，不包括读写寄存器时间

-  开方可选择两种模式

   -  只取整数（16位）

   -  包含小数（16+16位）

-  运算启动自动清除运算使能查询

-  提供运算进行标志和完成标志

-  支持有符号数和无符号数运算
