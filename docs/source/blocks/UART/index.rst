UART接口控制器 (UART)
=========================

概述
----
UART模块支持波特率配置，最高速度可达到模块时钟16分频。具备深度为8的FIFO，同时提供了多种中断供选择。

特性
----
-  支持标准的UART协议

-  支持全双工模式

-  支持波特率可配置

-  支持8位/9位数据格式选择

-  可配置的奇偶校验位

-  支持1位/2位停止位选择

-  支持波特率自动调整

-  深度为8字节的发送和接收FIFO

-  支持break操作自动检测

-  支持接收超时中断

-  支持发送/接收数据LSB/MSB选择

-  支持发送/接收数据电平反向
