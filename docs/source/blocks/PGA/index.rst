可编程增益运放 (PGA)
=========================

概述
----
SWM221内置3路可编程增益运放，可配置为运放模式。

特性
----
-  PGA模式支持1/5/10/20倍放大

-  输出电阻模式

-  输出可直接或通过buffer进入ADC通道

-  三个PGA共用一个内置输出偏置：1.2，1.8，2.25V，(VREF_ADC/2)
